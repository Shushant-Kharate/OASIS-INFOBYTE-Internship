# OASIS INFOBYTE Internship  
## Data Analytics & Machine Learning Projects

This repository contains all the tasks completed during my **Data Analytics & Machine Learning Internship at OASIS INFOBYTE**.  
The internship focused on applying data science concepts to **real-world datasets**, covering the complete workflow from **data preprocessing and analysis** to **machine learning model development and prediction**.

Each task is implemented using **Python and Jupyter Notebook** and demonstrates practical problem-solving skills relevant to industry applications.

### üîπ Task 1: Iris Flower Classification
- Built a **Logistic Regression** model to classify Iris flower species.
- Used sepal and petal measurements as features.
- Evaluated model performance using training and testing accuracy.
- Implemented **user input prediction** for real-time classification.

**Concepts Used:** Classification, Logistic Regression, Model Evaluation

### üîπ Task 2: Unemployment Rate Analysis (COVID-19 Impact)
- Analyzed **state-wise unemployment trends in India**.
- Studied the impact of **COVID-19** on employment rates.
- Visualized unemployment rate changes over time.
- Compared rural and urban unemployment patterns.

**Concepts Used:** Exploratory Data Analysis, Time Series Analysis, Data Visualization

---

### üîπ Task 3: Car Price Prediction
- Developed a **regression model** to predict car prices.
- Used features such as brand, fuel type, transmission, ownership, and usage.
- Handled categorical variables using encoding techniques.
- Enabled price prediction for new user inputs.

**Concepts Used:** Regression, Feature Engineering, Model Training

---

### üîπ Task 4: Email / SMS Spam Detection
- Built a **machine learning-based spam detection system**.
- Classified messages into **spam and non-spam**.
- Applied text preprocessing and feature extraction.
- Trained and evaluated the model for accurate classification.

**Concepts Used:** Natural Language Processing (NLP), Text Classification, ML Models

---

### üîπ Task 5: Sales Prediction Using Advertising Data
- Predicted product sales based on advertising spend.
- Used **TV, Radio, and Newspaper** advertisement budgets as features.
- Trained a **Linear Regression** model.
- Implemented prediction for custom advertising inputs.

**Concepts Used:** Regression Analysis, Business Analytics, Predictive Modeling
